# 光电子文献自动抓取系统

一个面向光电子方向的自动化论文追踪工具。

该项目用于根据用户配置的关键词、期刊白名单和主题范围，自动抓取过去 24 小时内新发表或新收录的相关论文，完成筛选、去重、开放获取检测，并以列表和邮件的形式推送结果。

## 项目目标

本项目希望解决光电子研究人员“信息源分散、更新频繁、人工检索耗时”的问题，构建一个轻量、合规、可持续运行的每日论文监控系统。

系统重点覆盖：

- arXiv 预印本
- Crossref 正式期刊元数据
- OpenAlex 主题补充检索
- Unpaywall 开放获取链接检测

## 核心特性

- 按关键词、同义词和缩写规则筛选相关论文
- 支持语义匹配增强，提高召回能力
- 按最近 24 小时窗口抓取新增论文
- 统一归一化三类数据源的论文元数据
- 优先以 DOI 去重，无 DOI 时使用标题加作者指纹
- 输出合法落地页链接和开放获取 PDF 链接
- 通过 SQLite 记录论文与推送日志，避免重复通知
- 支持 SMTP 邮件摘要推送

## 合规原则

本项目从设计上遵守以下边界：

- 只使用官方或开放 API，不抓取出版商网页 HTML
- 不接入任何破解下载源
- 不存储论文 PDF 文件本身，只提供合法链接
- 对需要的接口统一附带联系邮箱标识
- 遵守各数据源的请求速率限制

## 计划架构

整体流程如下：

```text
配置加载
  -> arXiv / Crossref / OpenAlex 抓取
  -> 元数据归一化
  -> 时间过滤
  -> 去重
  -> 关键词和语义匹配
  -> Unpaywall 增强
  -> 数据库存储
  -> 列表导出与邮件推送
```

## 适用场景

该项目适合以下需求：

- 跟踪光电子、光通信、硅光、超表面、量子光源等方向的新论文
- 为课题组或个人研究者生成每日论文快报
- 搭建内部研究情报监控工具
